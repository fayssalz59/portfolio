# Fayssal Zeggar

Data Engineer based in Toronto, focused on Snowflake-based ETL/ELT pipelines, SQL modeling, Python integration work, analytics enablement, and governed data access in regulated pharmaceutical manufacturing environments.

My professional experience is centered on building reliable data flows for manufacturing and enterprise systems, supporting analytics, AI, and ML initiatives, and making complex process data easier for technical users to access and trust.

My public GitHub projects show two complementary sides of my work: **TikTok Recipe Intelligence / Tastagram**, a Snowflake-centered data platform with LLM enrichment and a live recipe app, and **World Cup AI Platform**, an Azure-oriented data and ML pipeline with StatsBomb data, Parquet layers, FastAPI inference, Terraform, and tests.
